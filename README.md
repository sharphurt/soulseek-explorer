# Soulseek Discovery SPA

React SPA for searching tracks, polling Soulseek search results, initiating downloads, and monitoring active downloads in a persistent sidebar.

## Setup

```bash
npm install
npm run dev
```

The API defaults to `http://localhost:8080`. Override it with:

```bash
VITE_API_BASE_URL=http://localhost:8080
```

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```
